# productManager  
  基于Spring Cloud架构的微服务应用，并且前后端分离。  
  完成【商品管理】管理模块  

## 技术方案
  - **前端**
    - React
  - **服务器**
    - Spring Cloud Netflix
      - Eureka
      - Feign
      - Ribbon
## 目录结构
```text
 └─productManager
    ├─configs                   # 配置文件存放处
    ├─dbscripts                 # 数据库执行脚本
    ├─docs                      # 项目相关文档
    └─src                       # 项目源码
```
